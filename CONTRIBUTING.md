# Contributing

This constitution is released under CC0. Anyone may use, adapt, or build upon it.

## How to Contribute

1. **Report issues** — Open a GitHub issue describing the problem, affected article or section, and suggested fix.

2. **Propose amendments** — During the constitutional amendment window (Years 0–5), amendments may be proposed via pull request with a clear justification.

3. **Suggest companion standards** — New companion standards that operationalize constitutional duties may be proposed via issue or pull request.

## Guidelines

- Reference specific articles and sections
- Explain the problem before proposing a solution
- Constitutional text changes require stronger justification than companion standard changes
- Binding standards must not weaken, blur, or contradict the constitution
